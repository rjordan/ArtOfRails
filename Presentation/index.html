Chapter 1: Humble Beginnings

Who Built Ruby

Ruby's creator, Yukihiro "Matz" Matsumoto, blended parts of his favorite languages (Perl, Smalltalk, Eiffel, Ada, and Lisp) to form a new language that balanced functional programming with imperative programming.

Why Ruby

Ruby is designed with programmer happyness in mind. Matz has managed to blend elegance, simplicty, and power to create a fantastic language.

Object Oriented

EVERYTHING in Ruby is an object

"string".length => 6

1.class => Fixnum
3.14159.class => Float
        
nil.class => NilClass

Arrays, Hashes, and Symbols

Arrays, Hashes, and Symbols are all first order objects in Ruby. 

[1, "Two", 3.14].length => 3

{key1: 1, key2: "Two", key3: 3.14}.length => 3
or
{:key1=>1, :key2=>"Two", key3=>3.14}.length => 3

:symbol.object_id == :symbol.object_id

Iterators and Blocks


[1,2,3,4].each do |item|
  puts item
end
          

[4,2,1,3].sort { |item1, item2|  item2<=>item1 } => [4, 3, 2, 1]
          

sqr = lambda { |n| n*n }   
or
sqr = ->(n){n*n}

sqr.call(3) => 9

Classes and Modules


module MyNamespace
  class Program
    def run()
      puts "Running!"
    end
  end
end
p = MyNamespace::Program.new
p.run => "Running!"
          
Open for Extension
class String
  def foo
    "Foo#{self}"
  end
end

"Bar".foo => "FooBar"

Classes and Modules: Part Deux Ducks


class Dog
  def speak
    puts "Bark Bark"
  end
end

class Cat
  def speak
    puts "Meow"
  end
end

def SaySomething(animal)
  puts animal.speak
end

SaySomething(Cat.new) => "Meow"

Classes and Modules: Part Three Mixins


module Runnable
  def run
    puts "Running!"
  end
end

class Program
  include Runnable
end

p = Program.new
p.run => "Running!"
